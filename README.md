A distributed synchronization tool for monitoring Capybara populations across South American wetlands.

## Installation
```bash
pip install -r requirements.txt
```

## Security Note
As of v1.1, all API communication requires a valid CAPY_SYNC_TOKEN.
Please copy config/settings.yaml.example to config/settings.yaml and update your keys.
